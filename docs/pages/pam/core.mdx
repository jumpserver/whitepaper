---
title: 7.4 PAM 核心功能详解
---

覆盖身份鉴别、访问控制、会话审计、密钥/凭据托管与合规报表等能力。

<img src="/whitepaper/image052.png" alt="PAM 功能示意" width="800" />

要点：
- 强认证：MFA、基于风险的策略
- 最小权限：基于角色与审批
- 全量审计：命令、录屏、检索与归档

## 7.4.1 身份鉴别

- 支持密码、OTP、硬件令牌等多因素认证
- 登录策略：来源网段/时段/设备限制

## 7.4.2 访问控制

- RBAC/ABAC 组合授权，临时提权与审批
- 命令黑白名单、双人会话、会话水印

## 7.4.3 审计

- 命令结构化采集与检索
- 录屏高保真回放与导出
- 长期归档与保留期策略

## 7.4.4 策略示例（伪代码）

```yaml
login-policy:
  allowed-cidrs: ["10.0.0.0/8", "192.168.0.0/16"]
  time-window: "09:00-20:00"
  mfa-required: true

rbac:
  roles:
    - name: ops
      resources: ["ssh", "db"]
      actions: ["connect", "read", "exec"]
  approvals:
    - action: "exec"
      when: resource == "db"
      approvers: ["team-lead"]

command-control:
  blacklist: ["rm -rf /", ":(){ :|:& };:"]
  whitelist: ["systemctl status", "tail -f"]
```

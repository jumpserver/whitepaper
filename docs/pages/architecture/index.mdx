## 总览

JumpServer 以控制面/数据面解耦为核心思想，围绕身份鉴别、授权、会话代理与审计存储构建完整闭环，支持横向扩展与多租户隔离。

<img src="/whitepaper/image022.jpg" alt="架构总览图" width="800" />

## 模块说明

- 认证与授权（IAM）
- 资产与会话管理
- 审计与合规

## 设计原则

- 安全优先：强认证、最小权限、全链路审计
- 可扩展：组件无状态、分层解耦、横向扩展
- 可观测：指标、日志、追踪覆盖关键路径
- 易集成：开放 API 与主流目录/工单系统对接

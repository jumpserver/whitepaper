## 1.1 编写目的

在数字化转型与等保 2.0 的背景下，企业面临身份鉴别、访问控制与安全审计等系统性挑战。JumpServer 作为开源堡垒机，提供统一身份、细粒度授权、全量审计与自动化能力，兼顾传统环境与云原生场景，帮助企业达成合规、提升运维效率与可追溯性。

## 1.2 组织与维护

- 白皮书按章节展开，配套一页聚合与侧栏导航，便于检索与分享
- 后续随版本更新同步修订，保持与产品能力一致

## 1.3 企业价值

- 统一入口：整合主机/数据库/应用访问路径
- 最小权限：角色/审批/临时授权与隔离控制
- 全量审计：命令与录屏可检索、可归档、可追溯
- 易部署：在线/离线安装，高可用与分布式扩展

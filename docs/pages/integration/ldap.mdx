---
title: 6.1.5 LDAP 目录服务集成
---

支持与企业 LDAP/AD 集成，实现统一身份与分组同步。

<img src="/whitepaper/image060.jpg" alt="LDAP 集成截图" width="800" />

步骤概览：
1. 配置目录连接与 Base DN
2. 字段映射与组同步规则
3. 登录策略与 MFA 绑定

## 前置条件

- 可访问 LDAP/AD 控制器的网络连通性
- 具备查询所需范围的 Bind 账户（最小权限）

## 连接与范围

- 服务器：`ldaps://dc.example.com:636`（推荐 LDAPS）
- Base DN：`DC=example,DC=com`
- Bind DN：`CN=svc_ldap,OU=Service,DC=example,DC=com`

## 字段映射（示例）

| 本地字段 | 目录属性 |
| --- | --- |
| 用户名 | `sAMAccountName` |
| 显示名 | `displayName` |
| 邮箱 | `mail` |
| 组 | `memberOf` |

## 组同步与授权

- 选择同步范围：组织单位 OU 或过滤器（如 `(objectClass=person)`）
- 组到角色映射：`CN=DevOps,OU=Groups → 角色: 运维`
- 冲突策略：本地存在同名用户时的合并/忽略规则

## 登录与安全策略

- 启用 MFA：为同步用户配置 OTP/硬件令牌
- 登录限制：来源网段/时段/设备类型
- 风险处置：异常登录告警与临时冻结

## 故障排查

- 连接失败：检查证书链与端口放通
- 同步异常：验证过滤器与 Base DN 是否正确
- 属性缺失：确认字段映射与用户对象类
